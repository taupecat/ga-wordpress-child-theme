Why a Child Theme?

  • Allows us to learn WordPress development concepts without having to start from scratch.
  • We can customize a theme without fear of upgrading.
  • Frameworks often work under a child theme model.

Basics of a Child Theme

6 Wives, > 4 Children, No Grandchildren
  • style.css
    Requires specially-formatted comments to be recognized by WordPress.
  • functions.php
    Loaded before the parent's functions.php file.
  • Template Files
    If exists, replaces the parent theme's template file.
